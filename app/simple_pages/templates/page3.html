What is Pytest?

Pytest is a python based testing framework, which is used to write and execute test codes. In the present days of REST services, pytest is mainly used for API testing even though we can use pytest to write simple to complex tests, i.e., we can write codes to test API, database, UI, etc

Pytest can be performed from pycharm terminal by the following:

$ pytest --pylint --cov

                def test_context_variables_year(client):

                current_date_time = datetime.datetime.now()

                content = bytes(test_string, 'utf-8')

                assert content in response.data

Flask & Flask Blueprint

Flask is a micro web framework written in Python. It is classified as a microframework because it does not require particular tools or libraries. It has no database abstraction layer, form validation, or any other components where pre-existing third-party libraries provide common function

A Blueprint is a way to organize a group of related views and other code. Rather than registering views and other code directly with an application, they are registered with a blueprint. Then the blueprint is registered with the application when it is available in the factory function.
